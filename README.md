# Alien Avatars

<p align="center">
Generate unique SVG alien avatars from any seed string.<br/>
Works with React and Vue 3.<br/>
Built by Aliens.
</p>

</div>

## Why

Alien Avatars makes deterministic extraterrestrial profile pictures. Same input always produces the same outputeyes. Built originally for Nexus agent profiles. Everything runs client-side. No external image host to depend on. Works with React & Vue.

Zero runtime dependencies.

#### React

```bash

```

#### Vue

```bash
npm install @zenon-red/alien-avatars-vue
```
